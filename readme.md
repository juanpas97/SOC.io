# SOC.&#8203;io

SOC.&#8203;io is a Windows all-in-one tool for cybersecurity analysts.

## Description
Modules:
* URL Reputation
* File Analyzer

### Tech

SOC.&#8203;io uses:

* [.NET Framework](https://dotnet.microsoft.com/download/dotnet-framework/net472) - Framework for creating Windows applications.
